# RSS Feed Generator

This is an opinionated fork of (the ngl somewhat poorly maintained) [Olshansk/rss-feeds](https://github.com/Olshansk/rss-feeds).

I have removed many feeds and aim to add some I deem interesting.

## Feeds

All feeds below, bundled as an OPML file for one-click import into an RSS reader: [feeds.opml](https://raw.githubusercontent.com/tedzards509/ai-blogs-rss/refs/heads/main/feeds.opml)
